# AniSiren
AniSiren — Where anime gets dangerously addictive.


A seductive anime discovery web app — built to tempt you into your next obsession.

Dive into temptation. Stream anime that seduces your soul.

About the Project

AniSiren is a modern anime browsing platform that lets users search, filter, and explore thousands of anime titles with beautiful visuals and smooth UX. Inspired by Aniwatch-style sites, it focuses on discovery through powerful search, filtering, sorting, seasonal lists, and detailed anime pages.

Purpose
Help anime fans quickly find their next binge-worthy series with rich metadata, stunning posters, and intuitive navigation — all powered by real anime data.

Features

- Browse trending / top / seasonal anime
- Global search with debouncing
- Advanced filters: Genre, Year, Season, Type (TV/Movie/OVA), Rating, Status
- Sorting: Score, Popularity, Title, Start Date
- Infinite scroll or pagination
- Responsive dark UI with neon/siren accents
- Anime detail pages (synopsis, characters, trailer, episodes list)
- User favorites / watchlist (localStorage + later backend)
- Random anime button
- Theme switcher (dark/light)
- Recommendations based on genre

Technologies Used

- HTML
- CSS
- JAVASCRIPT
